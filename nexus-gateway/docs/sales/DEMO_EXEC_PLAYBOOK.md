# DEMO_EXEC_PLAYBOOK.md

## Objetivo ejecutivo

Mostrar como Nexus reduce riesgo operativo y acelera adopcion de agentes con control medible.

## Narrative (15-20 min)

1. Problema actual (3 min)
- agentes conectando APIs criticas sin gobierno uniforme

2. Solucion (4 min)
- Nexus como control plane: auth/policy/egress/secrets/audit

3. Evidencia (5 min)
- mostrar auditoria y bloqueos de riesgo
- mostrar dashboard de salud y volumen

4. Impacto negocio (4 min)
- menor riesgo de exfiltracion
- integraciones mas rapidas
- evidencia para security/compliance

5. Cierre (2 min)
- propuesta de POC de 2-4 semanas con KPI de exito

## KPI sugeridos para POC

- tiempo de integracion primer flujo
- % de requests bloqueados por policy/egress con riesgo real
- reducción de incidentes operativos ligados a tools
